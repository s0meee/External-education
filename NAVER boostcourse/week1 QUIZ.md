###### 1. 다음중 파이썬의 특징이 아닌 것은? 
a. 플랫폼 독립적
b. 인터프리터 언어
c. 객체 지향
d. 동적 타이핑 언어
e. 처음 자바로 구현되었음

> Answer : e.처음 자바로 구현 되었음 


###### 2. 파이썬에서 처리할 수 있는 기본 자료형 (primitive data types) 이 아닌 것은?
a. integer
b. float
c. var
d. string
e. boolean

> Answer :c. var

###### 3. 파이썬에서 아래 코드를 실행시켰을 시에 a의 타입은?
Code:

```sh
a = 10
float(a)
type(a)
```
a. <<class 'int'>>
b. <<class 'float'>>
c. <<class 'string'>>

> Answer : a. <<class 'int'>>
